# PRISM
PRISM: Presence Real-time Identification & Security Model

PRISM is a web-based, inclusive biometric attendance and authentication system designed to ensure accurate identity verification while preventing proxy attendance and fraud. The system integrates:

Face Recognition using pre-trained models (FaceNet / DeepFace) for embedding extraction and similarity comparison

Fingerprint Verification for Niqab-friendly or alternative authentication

Liveness Detection via eye blink, head movement, and facial motion checks

Intelligent Decision Logic using machine learning (Isolation Forest / Logistic Regression) on attendance logs for anomaly detection

PRISM emphasizes privacy, ethical AI, and modular design, storing only encrypted embeddings or templates rather than raw biometric data. The project demonstrates real-world applicability in universities, remote work, and secure web portals.
